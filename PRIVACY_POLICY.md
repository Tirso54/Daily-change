# Política de Privacidad - Daily Challenge

## Recopilación de datos
Daily Challenge recopila los siguientes datos:
- Desafíos generados localmente en tu dispositivo
- Preferencias de usuario (guardadas localmente)

## Uso de datos
- No compartimos tus datos con terceros
- No vendemos tu información
- Los datos se procesan únicamente en tu dispositivo

## APIs externas
- Utilizamos la API Gemini de Google para generar desafíos
- Google puede recopilar ciertos datos según su política de privacidad
- Ver: https://policies.google.com/privacy
